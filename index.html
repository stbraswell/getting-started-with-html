1) The Internet

The Internet is a collection of computers that talk to each other. The web is a collection of HTML docs. When a user clicks on a link, that link is essentially an address to an HTML file on a server. The browser translates that HTML file into a webpage that the user sees. HTTP is the main protocol of the web. HTML stands for Hypetext Markup Language.

2) Tags

Tags are instructions to the computer and are not seen outside of the code (e.g. on the webpage). Elements are the complete tag, so from opening to closing. These are also the "markup". Some tags need to be closed, if these types of tags are not closed, then the tag will apply to all text following the opening of the tag. Other tags do not need to be closed, these are called "Void Tags". Tags can also have attributes. These attributes give more dimension to a tag, such as linking a webpage to text.

3) Whitespace in HTML

Whitespace in HTML doesn't translate 1:1 on the webpage. Spaces seem to be 1:1, but multiple spaces or carrige returns do not. For example, this is what 10 spaces looks like " ".

4) Inline and Block

There are inline and block elements. Block elements seem to protect their contents by fencing them, like the paragraph element.
